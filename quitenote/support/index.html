QuiteNote Support

QuiteNote is a privacy utility for encrypting supported text before you share it. If something isn't working, this page is the first stop.

Best report format

  • Device model
  • iOS version
  • QuiteNote version
  • Main app or Messages extension
  • Repro steps and screenshots

1. Before opening an issue

Include the exact screen flow, expected vs actual behavior, and whether the problem appears in the main app or the Messages extension.

2. Common support topics

Language or localization

If QuiteNote remains in English after changing device language, fully close and reopen the app and the Messages extension.

Face ID or app lock

If QuiteNote relocks unexpectedly after biometric authentication, include the unlock flow and whether it happens after backgrounding the app.

Messages extension problems

If the Messages extension looks blank, opens slowly, or loses state, mention compact vs expanded mode and whether ciphertext or a key was already on the clipboard.

Clipboard or key import

If paste or key import fails, say whether you were pasting ciphertext, a public key, or scanning a QR code. If safe, include visible error text and screenshots.

3. Contact and issue reporting

The public support path currently points to GitHub Issues:

  • Open a GitHub issue
  • Read the privacy policy

4. What to mention for faster triage

  • Whether the issue happens every time or intermittently
  • Passphrase mode or trusted-contact mode
  • Whether the clipboard contained encrypted text or key material
  • Whether the problem started after an app or iOS update
